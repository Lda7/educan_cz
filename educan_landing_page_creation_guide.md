# Educan.cz – Landing Page Creation Guide

This document describes how to create a high-converting landing page for **educan.cz** (Vzdělávací centrum Educan). It is written as a practical build spec: goals, structure, copy blocks, UX/UI guidance, SEO, analytics, and a delivery checklist.

> Context: Educan presents itself as a modern education center offering language education plus broader learning (tutoring, teacher methodology, digital literacy and AI trainings), delivered **online & in-person** (individual and group). The current site navigation includes: Domů, Vzdělávací kurzy, Jazykové kurzy, Přípravné kurzy, Webináře, E‑learning, O nás, Kontakt.  
> Use this landing page to **focus attention on one primary offer** (or one campaign) and convert visitors into leads/enrollments.

---

## 1) Purpose and success criteria

### Primary purpose
Turn targeted traffic (ads, social, newsletters, partners) into:
- **Course inquiries** (lead form / call / email), and/or
- **Direct enrollment** (if the course catalog supports it), and/or
- **Newsletter sign-ups** (secondary).

### KPI targets (examples)
- Landing page conversion rate (CR): 3–10% (depends on traffic source)
- Cost per lead (CPL): set per campaign
- Form completion rate: > 60%
- Scroll depth: 50%+ reaching the main CTA block
- Mobile performance: LCP < 2.5s on 4G for core content

---

## 2) Choose one landing page “campaign”

A landing page should have **one main offer**. Pick one of these common Educan campaign types:

1. **AI / digital skills for teachers** (webinar or course)
2. **ÚP-supported digital courses (82% support)** (career / upskilling)
3. **Maturita prep** (math / English) – summer “2nd attempt” angle
4. **Language course enrollment** (kids/adults, Czech for foreigners, ECL, etc.)

> If you need multiple offers, create **separate landing pages** per offer, each with tailored messaging and keywords.

---

## 3) Target audience and messaging

### Audience examples
- **Teachers / educators** seeking safe, practical AI usage
- **Job seekers / career shifters** leveraging Úřad práce support
- **Students** preparing for maturita (and parents)
- **Adults** improving language skills; companies for corporate training

### Messaging framework
- **Outcome** (what changes after the course)
- **Proof** (certification, partners, testimonials, years, instructors)
- **Practicality** (hands-on, templates, lesson plans, real examples)
- **Delivery** (online / Praha / Brno / Znojmo, group/individual)
- **Safety** (GDPR, safe AI use, trusted methods)

## 4) Landing page information architecture (recommended)

A proven layout for Educan-style offers:

1. **Hero**
   - Headline (outcome)
   - Subheadline (who it’s for + format)
   - Primary CTA button (Enroll / Get info / Reserve a seat)
   - Secondary CTA (Call / Download syllabus)
   - Key trust badges (ECL, partners, “online & in-person”)

2. **Quick benefits**
   - 3–6 bullets: “What you’ll get”

3. **Course snapshot**
   - Duration (hours), format (online/on-site), dates, location(s), price, group size
   - Financing (if applicable – ÚP support, conditions)

4. **For whom**
   - Persona bullets: “This is for you if…” and “Not ideal if…”

5. **Curriculum / agenda**
   - Modules (accordion or short sections)
   - Deliverables (templates, exercises, recordings, certificates)

6. **Instructor / team**
   - Photo, short bio, credibility markers

7. **Social proof**
   - Testimonials, logos/partners, case examples

8. **FAQ**
   - Typical objections: time, level, certificate, refunds, financing steps

9. **Final CTA block**
   - Repeat offer + urgency or next start date
   - Form embedded + contact options

10. **Footer**
    - Company details, GDPR, terms, cookie settings link

## 5) Wireframe-level copy (templates)

Below are copy blocks you can adapt per campaign. Keep it short, specific, and outcome-oriented.

### 5.1 Hero (template)
**Headline:**  
„Naučte se [desired outcome] za [timeframe] – prakticky a bez zbytečné teorie.“

**Subheadline:**  
Pro [audience] | [online / Praha / hybrid] | [hours / sessions] | Začátečníci vítáni

**Primary CTA:**  
„Chci se přihlásit“ / „Získat informace“

**Secondary CTA:**  
„Zavolat“ / „Stáhnout osnovu“

**Trust line:**  
Certifikace / partneři / dlouhodobá praxe / ECL (if relevant)

### 5.2 Quick benefits (examples)
- Konkrétní postupy, které použijete hned v praxi
- Materiály a šablony ke stažení
- Bezpečné a odpovědné použití nástrojů (GDPR, školní prostředí)
- Online i prezenčně podle vašich možností

### 5.3 Course snapshot (example)
- **Forma:** Online / Praha  
- **Délka:** 16 hodin (4×4 hod.)  
- **Termín:** [date]  
- **Cena:** [price] (+ možnosti financování)  
- **Certifikát:** Ano/Ne, jaký  
- **Kapacita:** [number] účastníků

### 5.4 Financing (ÚP-supported) block
Explain in 3 steps:
1. Podmínky podpory (kdo může čerpat)
2. Jak podat žádost (link na ÚP / dokumenty)
3. Jak vám Educan pomůže (konzultace, podklady)

Add a clear disclaimer:
> Podmínky podpory se mohou lišit – vždy ověřte u Úřadu práce.

### 5.5 FAQ (seed list)
- Pro jakou úroveň je kurz?
- Co když se nemůžu zúčastnit části výuky?
- Dostanu záznam / materiály?
- Jak probíhá přihlášení a platba?
- Jak funguje podpora ÚP?
- Jaké jsou storno podmínky?

## 6) Design & UX guidelines

### Visual style
- Keep consistent with Educan brand (logo, typography, colors)
- Use **one primary accent color** for CTAs
- Avoid distractions: remove global nav if possible (or keep minimal)

### Mobile-first rules
- Hero CTA visible without scrolling
- Sticky bottom CTA on mobile (optional)
- Use large tap targets (44px+), readable body text (16px+)

### Conversion-focused UX
- Place a short form above the fold for lead-gen pages
- Reduce fields: name + email/phone + course interest
- Add microcopy for trust: “Ozveme se do 24 hodin (Po–Pá)”

---

## 7) SEO essentials (for landing pages)

Even for paid traffic, do basic SEO hygiene:
- Single **H1** with the main keyword + outcome
- Use “Course / Webinar” keyword in Title and meta description
- Add FAQ schema (if possible) for organic visibility
- Optimize images (WebP), use descriptive alt text
- Internal linking: from relevant category page(s) to the landing page

**Example title:**  
AI pro učitele | Praktický webinar (online) – Educan

---

## 8) Tracking, analytics, and consent

### Events to track
- Page view
- CTA clicks (primary and secondary)
- Form start / form submit
- Phone click (tel:)
- Email click (mailto:)
- Scroll depth (25/50/75/100)

### UTM handling
Preserve UTMs through the form submission (store in hidden fields) to attribute leads.

### Consent & privacy
- Respect cookie consent; fire marketing tags only after consent
- Link GDPR policy in the footer and near the form (short line is enough)

---

## 9) Implementation options

### Option A: WordPress landing page (fastest)
If educan.cz runs on WP (common for this type of site):
- Create a new page template “Landing – No Header”
- Use a page builder or custom blocks (Gutenberg) for sections
- Ensure performance: limit heavy sliders and large images

### Option B: Dedicated landing micro-site (for campaigns)
- Static site (Next.js / Astro) + simple form backend
- Best performance and A/B testing control

### Forms
- Integrate with email/CRM: HubSpot / Mailchimp / Brevo / custom SMTP
- Ensure spam protection: honeypot + reCAPTCHA (if needed)

## 10) Content checklist

### Must-have
- Clear offer + outcome

- Pricing + financing details (if relevant)

- Testimonials or proof (partners, certifications, references)

- “What you will create” deliverables gallery
- Countdown to next cohort start

## 11) Launch checklist

### Pre-launch
- [ ] Mobile layout verified (iOS + Android)
- [ ] Page speed tested (LCP, CLS)
- [ ] Form delivers to the right inbox/CRM
- [ ] Thank-you page configured + conversion tracking
- [ ] UTM capture verified

- [ ] Check conversion rate by source
- [ ] Review form drop-offs
- [ ] Heatmap/session recordings (if used)
- [ ] Run one A/B test: headline or CTA copy
